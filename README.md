# Swaraj Portfolio

This portfolio is ready to deploy on Netlify.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Netlify deploy settings

- Build command: `npm run build`
- Publish directory: `dist`

Because `netlify.toml` is included, Netlify should detect these settings automatically after you connect the repository.
